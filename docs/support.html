Parenting Timer

Support & Documentation — v1.2

Overview

Parenting Timer is a countdown timer and stopwatch with an adjustable speed control, designed for parents who want a little flexibility when negotiating time with children. You can agree on a time together, then secretly run the timer faster or slower than real time.

Basic Use

The app has two modes, selectable at the top: Countdown and Stopwatch. Each mode has its own independent settings.

Countdown

Set the duration using the +/− buttons or by tapping the time display to type directly. Tap Start to begin. The progress bar drains as time elapses. When it reaches zero, the alarm sounds.

Stopwatch

Tap Start to begin counting up. The progress bar fills up to the target duration. Tap Stop to pause, or Reset to start over.

The Hidden Speed Control

Tap the word "Timer" in the app title three times to reveal the speed control panel. This panel is not visible during normal use.

The speed multiplier controls how fast the timer runs relative to real time:

  • 1.0× — normal speed (real time)
  • 2.0× — twice as fast (a 10-minute countdown finishes in 5 real minutes)
  • 0.5× — half speed (a 10-minute countdown takes 20 real minutes)

The time displayed always counts in "timer minutes", not real minutes. A 10-minute countdown at 2× speed will count down from 10:00 to 0:00, but do so in 5 real minutes.

Speed Modes

Two speed modes are available in the hidden panel:

Ratio mode

You set the speed multiplier directly. The timer runs at that fixed multiple of real time for the entire duration.

Fixed time mode (countdown only)

You set both the displayed countdown duration and the real time you want it to take. The app calculates the required speed automatically. For example: display 10 minutes, finish in 8 real minutes → speed is automatically set to 1.25×.

This is useful for setting the speed before negotiating with children — decide on the real duration you want, open the hidden panel, set both values, then close the panel and show the child the displayed time.

Realtime When Visible

When this option is enabled (in the hidden panel), the timer runs at normal speed (1×) while you are looking at the screen, and makes up any difference in the background.

This means a child watching the timer will see it tick at normal pace. When the screen is off or the app is in the background, the timer catches up toward where it should be according to the target speed.

Max background speed

This sets the maximum speed the timer will use during background catch-up. It must be at least as far from 1× as the target speed. For example, if your target speed is 2×, the max background speed must be at least 2×.

The catch-up algorithm takes a conservative approach: on returning to the foreground, the timer advances to whichever position is smaller — either where it would be at max background speed, or where it ideally should be. It never jumps ahead of the ideal target.

Alert Settings

Tap the bell icon to configure what happens when the timer finishes:

  • Sound — plays the selected alert tone
  • Vibrate — vibrates the device (Android and iPhone only)

Tap the tone name to choose from a list of available sounds. The alert plays once when the timer finishes.

Frequently Asked Questions

Does the timer work when the screen is off?

Yes. The timer continues in the background and the alarm will fire even if the screen is off.

Does it work offline?

Yes, completely. The app requires no internet connection.

Is my data private?

The app collects no data other than your saved settings, which stay on your device. See the Privacy Policy for full details.

Why does the timer sometimes seem slightly off when I unlock my phone?

If "Realtime when visible" is enabled, the timer may be mid-catch-up when you return. It will settle to the correct position shortly. If the option is disabled, the timer runs at the configured speed regardless of screen state.
