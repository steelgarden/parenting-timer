Help & Documentation

Basic Use

Select Countdown or Stopwatch at the top. Each mode has its own independent settings.

Countdown

Set the duration using the +/− buttons or by tapping the time display to type directly. Tap Start to begin. The progress bar drains as time elapses. When it reaches zero, the alarm sounds.

Stopwatch

Tap Start to begin counting up. The progress bar fills up to the target duration. Tap Stop to pause, or Reset to start over.

The Hidden Speed Control

Tap the word "Timer" in the app title three times to reveal the speed control panel. This panel is not visible during normal use.

The speed multiplier controls how fast the timer runs relative to real time:

  • 1.0× — normal speed
  • 2.0× — twice as fast (10 min countdown finishes in 5 real minutes)
  • 0.5× — half speed (10 min countdown takes 20 real minutes)

The display always counts in "timer minutes". At 2×, a 10:00 countdown reaches 0:00 in 5 real minutes.

Speed Modes

Ratio mode

Set the speed multiplier directly. The timer runs at that fixed multiple of real time.

Fixed time mode (countdown only)

Set both the displayed duration and the real time you want it to take. The speed is calculated automatically. For example: display 10 min, finish in 8 real min → speed set to 1.25×.

Useful for setting the speed before negotiating with children — decide the real duration, configure it in the hidden panel, then close the panel and show the child the displayed time.

Realtime When Visible

When enabled, the timer runs at normal speed (1×) while you are looking at the screen, and catches up in the background.

A child watching the timer will see it tick at normal pace. When the screen is off, the timer catches up toward the target speed.

Max background speed

The maximum speed used during background catch-up. Must be at least as far from 1× as the target speed. The catch-up never jumps ahead of the ideal target position.

Alert Settings

Tap the bell icon to configure alerts when the timer finishes:

  • Sound — plays the selected alert tone once
  • Vibrate — vibrates the device

Frequently Asked Questions

Does the timer work when the screen is off?

Yes. The timer continues in the background and the alarm will fire even if the screen is off.

Does it work offline?

Yes, completely. The app requires no internet connection.

Is my data private?

The app collects no data other than your saved settings, which stay on your device.

Why does the timer sometimes seem slightly off when I unlock my phone?

If "Realtime when visible" is enabled, the timer may be mid-catch-up when you return. It will settle to the correct position shortly.
